## 레포지토리 용도

이 레포지토리는 오픈소스 SW의 수업 자료, 필기, 과제 코드 등을 저장하고 관리하는 목적을 가지고 있다. <br>주된 용도는 :
- 수업 필기 
- 과제 관리
- 수업자료 보관

## 폴더 설명

-  `수업과제/`: 과제를 위해 작성한 파일이 저장된 폴더이다.
- `수업필기/`: 매주 수업 필기를 저장한 폴더이다.
- `수업실습/`: 수업 중 실습했던 내용이 저장된 폴더이다.
- `수업자료/`: 수업이나 실습에 필요한 자료를 저장한 폴더이다.
